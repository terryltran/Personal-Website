Greenhouse Heating System

Technologies Utilized: Arduino, Fusion 360, Altium

Collaborated with 3 other peers to design and create a greenhouse heating system that runs on wind power, solar power, and electrical power. The system is an eco-friendly system that uses warm water as the heating medium and is circled through a closed-loop pipe system. Temperature is measured via temperature sensors. Pump and water heater are switched on and off by relays. Users can determine the minimum/desired temperature of the greenhouse via a 4x4 keypad that is attached to an Arduino Mega.

Arduino Bluetooth Car

Technologies Utilized: Arduino, Bluetooth

Collaborated with 1 peer to design and create an Arduino car that is controlled via Bluetooth using HC-05 Bluetooth modules. The system utilized two Arduino devices (Uno and Mega) to collect, send, and transfer sensor data from the Uno on the car to the Mega (then printed on an Arduino serial monitor) via Bluetooth. The car’s movement controlled via Bluetooth using an Android phone app and an HC-05 module connected to the Arduino Uno.

The Jolly Banker

Technologies Utilized: C++

Reads commands from a file on command prompt, which are put in a queue. Users can open an account, deposit and withdraw money, transfer money between funds or another person who uses the bank, and also look at the history of their account (specific fund or entire history). Accounts are kept in a binary search tree and sorted by the ID number chosen by the user.
